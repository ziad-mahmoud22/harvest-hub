

### Setup

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/ziad-mahmoud22/harvest-hub.git
    cd harvest-hub
    ```
2.  **Run the Backend:**
    *   Navigate to the backend folder: `cd backend`
    *   Install dependencies: `pip install -r requirements.txt`
    *   run the app: `python app.py`
